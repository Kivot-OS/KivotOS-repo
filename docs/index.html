‚ö° Quick Setup

Add KivotOS repository to your Debian/Ubuntu system:

1. Add GPG Key

curl -fsSL https://dungdinhmanh.github.io/KivotOS-repo/apt/pubkey.gpg | sudo gpg --dearmor -o /usr/share/keyrings/kivotos.gpg

2. Add Repository

echo "deb [signed-by=/usr/share/keyrings/kivotos.gpg] https://dungdinhmanh.github.io/KivotOS-repo/apt/ trixie main" | sudo tee /etc/apt/sources.list.d/kivotos.list

3. Update & Install

sudo apt update
sudo apt install yazi hellwal
